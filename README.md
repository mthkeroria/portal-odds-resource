# Portal Odds Resource

一个用于归档与发布独立 HTML 页面的通用仓库。本仓库不针对特定网站或域名，仅作为独立页面资源的存储与分发空间。

## 项目简介

`portal-odds-resource` 旨在提供一个轻量、简洁的 HTML 页面归档方案。每个页面均为独立、自包含的 HTML 文件，可单独访问或嵌入使用。仓库本身不包含后端逻辑或动态内容，仅用于静态资源的存放与版本管理。

## 目录结构

```
.
├── index.html          # 仓库入口页面（可选）
├── pages/              # 归档的独立 HTML 页面
│   ├── example-page.html
│   └── ...
├── assets/             # 共用资源（CSS、JS、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- `pages/`：存放所有独立 HTML 页面，每个文件可独立访问。
- `assets/`：存放页面共用的样式、脚本或图片资源，减少重复文件。

## 页面归档说明

- 所有页面均为静态 HTML，不依赖外部数据库或服务端运行环境。
- 每个页面应尽量自包含，如需共用资源，建议放置于 `assets/` 目录下。
- 页面命名建议使用英文小写与连字符（如 `example-page.html`），以保持 URL 整洁。
- 归档页面不保证与任何特定网站或服务关联，仅为独立内容展示。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接使用浏览器打开 `pages/` 目录下的任意 HTML 文件即可预览。
3. 若需部署至 Web 服务器，将整个仓库目录上传至服务器根目录或子目录即可。
